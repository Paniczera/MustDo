🚧 Em desenvolvimento
  🚀 MustDo!
  
  Aplicação web para gerenciamento de tarefas diárias, com níveis de prioridade personalizáveis.
  
  📌 Sobre o projeto
  
  Esse projeto foi desenvolvido com o objetivo principal de por em prática minhas habilidades em desenvolvimento web.
  
  
  🛠️ Tecnologias utilizadas até o momento:
  HTML5
  CSS3
  JavaScript
  React (futuramente)
  Git & GitHub
  
  ⚙️ Funcionalidades
  ✅ Adicionar tarefas
  ⏳ Remover tarefas
  ⏳ Marcar como concluída
  ⏳ Definir grau de prioridade
  ⏳ Persistência de dados
  
  💻 Como rodar o projeto
  # Clone o repositório
  git clone https://github.com/Paniczera/MustDo.git
  
  # Acesse a pasta
  cd MustDo
  
  # Abra o arquivo index.html
